# Recomendify
## Recomedify es un programa que lee una base de datos de spotify para dar:
* Las canciones más importantes
* Una recomendación en base a un listado de canciones
* Un ciclo n de canciones utilizando un dfs
* El camino más corto entre dos canciones

### Además, se implementa un grafo en Python, y se usa el algoritmo de **PageRank** para determinar las canciones más importantes.
